# Checklists

## Add a new client
- Create client
- Add contacts
- Add domains + providers
- Add services + renewal dates
- Add assets/links (admin, repo, analytics)
- Add vault pointers (do NOT paste secrets)
- Create project + initial tasks

## Before a DNS change
- Confirm current NS + records
- Screenshot/export zone
- Note rollback plan
- Schedule low-traffic window
- Log decision in Decisions

## Before deploy
- Staging smoke test
- Backup (where relevant)
- Note release in timeline
